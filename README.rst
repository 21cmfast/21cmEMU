An emulator of 21cmFAST summaries.

* Free software: MIT license

Features
========

Emulate the following summary statistics:
    * 21-cm power spectrum
    * 21-cm global brightness temperature
    * IGM spin temperature
    * Neutral fraction
    * :math:`\\tau_e`
    * UV luminosity functions
